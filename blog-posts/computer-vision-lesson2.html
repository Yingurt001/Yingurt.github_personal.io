Learning Topics

1. Understanding Convolution Operation

  • Mathematical definition and principles of convolution
  • Concept and role of kernel (filter)
  • Kernel sliding across image to compute weighted sums
  • 1 kernel ‚Üí 1 feature map, 64 kernels ‚Üí 64 feature maps
  • Kernel size (typically odd sizes like 3√ó3 or 5√ó5)

2. Defining Convolution in TensorFlow

  • Using layers.Conv2D
  • Meaning of filters parameter (how many kernels to learn)
  • Role of kernel_size parameter
  • Difference between Keras layers and TensorFlow low-level operations
  • Using tf.nn.conv2d

3. Understanding Feature Maps (Activation Maps)

  • Feature map generation process
  • Each scan position outputs one number
  • All output values form a feature map
  • Feature maps show where features appear
  • Activation values before and after activation function

4. Role of ReLU Activation Function

ReLU is defined as:

$$\text{ReLU}(x) = \max(0, x)$$
  • ReLU acts as a feature detector
  • Keeps positive responses (feature found)
  • Suppresses negative values (feature not found)
  • Importance of ReLU in feature detection

5. Basic Structure of Convolutional Neural Networks

  • Conv2D layer (feature extraction)
  • ReLU activation (feature detection)
  • Max Pooling layer (feature compression)
  • These three steps constitute the feature extraction pipeline

6. Feature Extraction Workflow

The complete feature extraction process consists of three steps:

  1. Filter: Kernel scans the image, computes weighted sums
  2. Detect: ReLU keeps positive evidence of features
  3. Condense: Max Pooling reduces spatial size and enhances strong features

7. Manually Defining Convolution Kernels

  • Creating custom kernels using tf.constant
  • Understanding how sum of kernel values affects output brightness
  • Different types of kernels (edge detection, blur, sharpen)
  • How kernels modify images
